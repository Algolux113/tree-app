# tree-app
1. Install npm packages:
```npm install```
2. Run application:
```npm run dev```
3. Kill chrome process:
```TASKKILL /F /IM chrome.exe```
4. Run chrome without security:
```chrome.exe  --disable-site-isolation-trials --disable-web-security --user-data-dir="C:\lol"```
5. Open application on:
```http://localhost:8081/```
